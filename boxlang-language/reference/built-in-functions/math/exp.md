# Exp

Calculates the exponent whose base is e that represents a number.

## Method Signature

```
Exp(number=[numeric])
```

### Arguments

| Argument | Type      | Required | Description                               | Default |
| -------- | --------- | -------- | ----------------------------------------- | ------- |
| `number` | `numeric` | `true`   | The number to calculate the exponent for. |         |
