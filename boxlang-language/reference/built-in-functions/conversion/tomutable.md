# ToMutable

Convert an array, struct or query to its mutable counterpart.

## Method Signature

```
ToMutable(value=[any])
```

### Arguments

| Argument | Type  | Required | Description                            | Default |
| -------- | ----- | -------- | -------------------------------------- | ------- |
| `value`  | `any` | `true`   | The array, struct or query to convert. |         |
