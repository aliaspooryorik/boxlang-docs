# ExecutorGet

Get an executor by name

## Method Signature

```
ExecutorGet(name=[string])
```

### Arguments

| Argument | Type     | Required | Description                      | Default |
| -------- | -------- | -------- | -------------------------------- | ------- |
| `name`   | `string` | `true`   | The name of the executor to get. |         |
