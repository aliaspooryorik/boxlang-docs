# BitNot

Performs a bitwise logical NOT operation.

## Method Signature

```
BitNot(number=[integer])
```

### Arguments

| Argument | Type      | Required | Description                    | Default |
| -------- | --------- | -------- | ------------------------------ | ------- |
| `number` | `integer` | `true`   | Numeric value for bitwise NOT. |         |
