# SetLocale

Sets the current request-level locale.

## Method Signature

```
SetLocale(locale=[string])
```

### Arguments

| Argument | Type     | Required | Description                                    | Default |
| -------- | -------- | -------- | ---------------------------------------------- | ------- |
| `locale` | `string` | `true`   | The locale ISO directive, common name or alias |         |
