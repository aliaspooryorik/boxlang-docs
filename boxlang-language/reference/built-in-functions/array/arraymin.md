# ArrayMin

Return length of array

## Method Signature

```
ArrayMin(array=[any])
```

### Arguments

| Argument | Type  | Required | Description                     | Default |
| -------- | ----- | -------- | ------------------------------- | ------- |
| `array`  | `any` | `true`   | The array to get min value from |         |
