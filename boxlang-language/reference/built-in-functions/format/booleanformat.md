# BooleanFormat

Returns the value formatted as a boolean string

## Method Signature

```
BooleanFormat(value=[any])
```

### Arguments

| Argument | Type  | Required | Description                                                | Default |
| -------- | ----- | -------- | ---------------------------------------------------------- | ------- |
| `value`  | `any` | `true`   | The value to cast as a boolean and return the string value |         |
