# IsNull

Determine whether a given value or variable reference is null.

## Method Signature

```
IsNull(object=[any])
```

### Arguments

| Argument | Type  | Required | Description             | Default |
| -------- | ----- | -------- | ----------------------- | ------- |
| `object` | `any` | `true`   | Value to test for null. |         |
