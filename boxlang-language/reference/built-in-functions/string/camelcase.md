# CamelCase

Convert a string to camel case

## Method Signature

```
CamelCase(string=[string])
```

### Arguments

| Argument | Type     | Required | Description       | Default |
| -------- | -------- | -------- | ----------------- | ------- |
| `string` | `string` | `true`   | The target string |         |
