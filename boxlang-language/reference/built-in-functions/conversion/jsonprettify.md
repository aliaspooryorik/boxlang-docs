# JSONPrettify

Prettifies a JSON string.

## Method Signature

```
JSONPrettify(var=[string])
```

### Arguments

| Argument | Type     | Required | Description                  | Default |
| -------- | -------- | -------- | ---------------------------- | ------- |
| `var`    | `string` | `true`   | The JSON string to prettify. |         |
