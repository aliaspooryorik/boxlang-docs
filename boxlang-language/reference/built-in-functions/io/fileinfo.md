# FileInfo

Returns a struct of file information.

Different values are returned for FileInfo and GetFileInfo

## Method Signature

```
FileInfo(file=[any])
```

### Arguments

| Argument | Type  | Required | Description                                       | Default |
| -------- | ----- | -------- | ------------------------------------------------- | ------- |
| `file`   | `any` | `true`   | The filepath or file object to retrieve info upon |         |
