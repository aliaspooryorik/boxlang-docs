# StructKeyList

Get keys of a struct as a string list

## Method Signature

```
StructKeyList(structure=[structloose], delimiter=[string])
```

### Arguments

| Argument    | Type     | Required | Description | Default |
| ----------- | -------- | -------- | ----------- | ------- |
| `structure` | `struct` | `true`   |             |         |
| `delimiter` | `string` | `false`  |             | `,`     |
