# QueryColumnArray

This function returns the column array of a query.

## Method Signature

```
QueryColumnArray(query=[query])
```

### Arguments

| Argument | Type    | Required | Description                            | Default |
| -------- | ------- | -------- | -------------------------------------- | ------- |
| `query`  | `query` | `true`   | The query to get the column names from |         |
