# Acos

Returns the arccosine (inverse cosine) of a number

## Method Signature

```
Acos(number=[double])
```

### Arguments

| Argument | Type     | Required | Description                              | Default |
| -------- | -------- | -------- | ---------------------------------------- | ------- |
| `number` | `double` | `true`   | The number to calculate the arccosine of |         |
