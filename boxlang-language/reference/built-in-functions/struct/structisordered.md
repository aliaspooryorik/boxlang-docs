# StructIsOrdered

Tests whether a struct is ordered ( e.g.

linked )

## Method Signature

```
StructIsOrdered(struct=[structloose])
```

### Arguments

| Argument | Type     | Required | Description                          | Default |
| -------- | -------- | -------- | ------------------------------------ | ------- |
| `struct` | `struct` | `true`   | The struct to test for a linked type |         |
