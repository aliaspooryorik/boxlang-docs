# IsZipFile

Verifies if the incoming file absolute path is a zip file or not

## Method Signature

```
IsZipFile(path=[string])
```

### Arguments

| Argument | Type     | Required | Description                             | Default |
| -------- | -------- | -------- | --------------------------------------- | ------- |
| `path`   | `string` | `true`   | The absolute path of the file to verify |         |
