# Abs

Returns the absolute value of a number

## Method Signature

```
Abs(value=[numeric])
```

### Arguments

| Argument | Type      | Required | Description                                | Default |
| -------- | --------- | -------- | ------------------------------------------ | ------- |
| `value`  | `numeric` | `true`   | The number to return the absolute value of |         |
