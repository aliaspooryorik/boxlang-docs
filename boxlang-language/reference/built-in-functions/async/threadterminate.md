# ThreadTerminate

Terminates the thread specified by the threadName parameter.

## Method Signature

```
ThreadTerminate(threadName=[string])
```

### Arguments

| Argument     | Type     | Required | Description                          | Default |
| ------------ | -------- | -------- | ------------------------------------ | ------- |
| `threadName` | `string` | `true`   | The name of the thread to terminate. |         |
