# XMLGetNodeType

Get XML values according to given xPath query

## Method Signature

```
XMLGetNodeType(XMLNode=[XML])
```

### Arguments

| Argument  | Type  | Required | Description             | Default |
| --------- | ----- | -------- | ----------------------- | ------- |
| `XMLNode` | `XML` | `true`   | An XML DOM object node. |         |
