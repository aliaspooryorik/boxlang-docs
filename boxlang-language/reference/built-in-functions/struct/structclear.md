# StructClear

Clear all items from struct

## Method Signature

```
StructClear(structure=[modifiableStruct])
```

### Arguments

| Argument    | Type               | Required | Description          | Default |
| ----------- | ------------------ | -------- | -------------------- | ------- |
| `structure` | `modifiableStruct` | `true`   | The struct to clear. |         |
