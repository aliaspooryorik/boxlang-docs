# ExecutorHas

Verify if an executor exists

## Method Signature

```

```

### Arguments

| Argument | Type     | Required | Description                         | Default |
| -------- | -------- | -------- | ----------------------------------- | ------- |
| `name`   | `string` | `true`   | The name of the executor to verify. |         |
